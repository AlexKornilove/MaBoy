# Быстрый старт

## Запуск приложения

1. Откройте терминал в папке проекта
2. Запустите сервер:
   ```bash
   node Server.js
   ```
   или
   ```bash
   npm start
   ```
3. Откройте браузер и перейдите по адресу: **http://localhost:3000**

## Первое использование

1. При первом запуске данные загрузятся из файла `data/collection.json`
2. Вы можете кликать по карточкам, чтобы отмечать элементы как собранные
3. Используйте кнопку "⚙ Редактирование" для добавления новых элементов
4. Все изменения автоматически сохраняются в файл

## Добавление изображений

Поместите изображения в соответствующие папки:
- Варфреймы: `img/frames/название.png`
- Оружие: `img/weapons/название.png`

При добавлении элемента укажите путь к изображению в формате: `img/frames/название.png`

## Примечания

- Сервер должен быть запущен для работы приложения
- Данные сохраняются автоматически при каждом изменении
- Для остановки сервера нажмите `Ctrl+C` в терминале
